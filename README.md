# Message Queues Examples

This repository contains practical examples of working with message queue systems using Node.js.

## Contents

- **Kafka/** - Apache Kafka producer and consumer examples using KafkaJS
- **RabbitMQ/** - RabbitMQ producer and consumer examples using amqplib

## Prerequisites

- Node.js installed
- pnpm package manager
- Docker and Docker Compose (for running message brokers locally)

## Getting Started

Navigate to each directory (Kafka or RabbitMQ) to find specific implementation examples and instructions.
